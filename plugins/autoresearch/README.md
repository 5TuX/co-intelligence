# autoresearch

Iterative code-driven investigation. Write code, run it, analyze results, refine the hypothesis, repeat.

## Credits

Inspired by [karpathy/autoresearch](https://github.com/karpathy/autoresearch) — Andrej Karpathy's framework for AI agents running automated research (originally demonstrated on single-GPU nanochat training). This plugin implements the methodology as a Claude Code skill; no code from the upstream repo is included. Packaged and maintained as part of the `co-intelligence` marketplace by 5TuX.
